Document Structure Tags

9 tags
<!DOCTYPE html> Declaration
Tells the browser this is an HTML5 document. Must be the very first line.
<!DOCTYPE html>
<html> Block
The root element that wraps all HTML content. The lang attribute sets the language.
<html lang="en">
  ...
</html>
<head> Block
Contains metadata: title, links to stylesheets, scripts, and SEO tags. Not visible on page.
<head>
  <meta charset="UTF-8">
  <title>My Page</title>
</head>
<body> Block
Contains all visible page content: text, images, links, forms, etc.
<body>
  <h1>Hello World</h1>
</body>
<title>
Sets the browser tab title and is used by search engines.
<title>My Awesome Website</title>
<meta> Void
Provides metadata like charset, viewport, description, and author.
<meta charset="UTF-8">
<meta name="viewport"
      content="width=device-width,
               initial-scale=1.0">
<meta name="description"
      content="Page description">
<link> Void
Links external resources like CSS stylesheets or favicons.
<link rel="stylesheet"
      href="style.css">
<link rel="icon"
      href="favicon.ico">
<style>
Embeds internal CSS rules directly inside the HTML document.
<style>
  body { background: #fff; }
  h1   { color: navy; }
</style>
<base> Void
Sets the base URL for all relative links and targets in a document.
<base href="https://example.com/"
      target="_blank">

10 tags

Scripting Tags

3 tags

Embeds or links to JavaScript. Use defer or async for performance.

CSS: Box Model Properties

width / height / min-* / max-*
Values: px · % · em · rem · vw · vh · auto · fit-content · max-content · min-content
Controls the dimensions of an element's content box.
div {
  width: 100%;
  max-width: 1200px;
  min-height: 300px;
  height: auto;
}
margin
Shorthand: top right bottom left · auto · negative values allowed
Space outside the element border. Use auto to center block elements.
/* 4-value shorthand: top right bottom left */
div { margin: 10px 20px 10px 20px; }

/* Center horizontally */
div { width: 600px; margin: 0 auto; }

/* Individual sides */
div { margin-top: 16px; margin-bottom: 8px; }
padding
Shorthand: top right bottom left · no negative values
Space inside the border, between content and border.
/* 2-value: vertical horizontal */
div { padding: 16px 24px; }

/* 4-value */
div { padding: 8px 12px 8px 12px; }

/* Individual */
div { padding-left: 20px; }
border
border-width · border-style · border-color · border-radius
Shorthand for width, style, and color of an element's border.
div {
  border: 2px solid #38bdf8;
  border-radius: 8px;
  border-top: 4px solid red;

  /* Individual sides */
  border-left: 3px dashed green;
  border-bottom: 1px dotted gray;
}
solid
dashed
dotted
double
box-sizing
content-box (default) · border-box
border-box includes padding and border in the element's total width/height. Always use it.
*, *::before, *::after {
  box-sizing: border-box; /* Best practice */
}

/* content-box (default):
   width = content only
   border-box:
   width = content + padding + border */
display
block · inline · inline-block · flex · grid · none · contents · table
Controls how an element is rendered in the document flow.
span   { display: block; }        /* Now takes full width */
div    { display: inline; }       /* Now flows inline */
button { display: inline-block; } /* Both properties */
.hide  { display: none; }         /* Removed from layout */
.flex  { display: flex; }
.grid  { display: grid; }
position
static · relative · absolute · fixed · sticky
Positions elements within their container or viewport.
.relative { position: relative; top: 10px; left: 5px; }
.absolute { position: absolute; top: 0;    right: 0; }
.fixed    { position: fixed;    bottom: 20px; right: 20px; }
.sticky   { position: sticky;   top: 0; }

/* z-index controls stacking */
.on-top   { position: relative; z-index: 10; }
overflow
visible · hidden · scroll · auto · clip
Controls what happens when content overflows an element's box.
div { overflow: hidden; }     /* Clip content */
div { overflow: auto; }       /* Scroll only when needed */
div { overflow: scroll; }     /* Always show scrollbars */
div { overflow-x: hidden;     /* Individual axes */
      overflow-y: auto; }

CSS: Typography Properties

font-family
Font name · generic (serif / sans-serif / monospace / cursive / fantasy)
Sets the typeface. Always end with a generic family as fallback.
body {
  font-family: 'Roboto', Arial, sans-serif;
}
code {
  font-family: 'JetBrains Mono', monospace;
}
font-size / font-weight / font-style
px · em · rem · % · vw · 100–900 · normal · bold · italic
Core typographic sizing and styling properties.
h1 { font-size: 2.5rem;  }  /* 40px at base 16px */
p  { font-size: 1rem;    }  /* 16px */

.bold   { font-weight: 700; }
.thin   { font-weight: 300; }
.italic { font-style: italic; }
line-height / letter-spacing / word-spacing
px · em · unitless (recommended) · normal
Controls vertical and horizontal spacing between text elements.
p {
  line-height: 1.7;          /* Recommended for body */
  letter-spacing: 0.02em;   /* Slight tracking */
  word-spacing: 2px;
}

h1 {
  line-height: 1.1;          /* Tighter for headings */
  letter-spacing: -0.03em;  /* Negative for large text */
}
text-align / text-decoration / text-transform
left · center · right · justify · underline · none · uppercase · capitalize
Alignment, decoration, and case transformation of text.
h2 { text-align: center; }
a  { text-decoration: none; }    /* Remove underline */
.caps   { text-transform: uppercase; }
.title  { text-transform: capitalize; }
.del    { text-decoration: line-through; }
text-shadow / text-overflow
h-offset v-offset blur color · ellipsis · clip
Adds shadow to text. text-overflow:ellipsis shows "…" for clipped text.
h1 {
  text-shadow: 2px 2px 4px rgba(0,0,0,0.3);
  /* Multiple shadows */
  text-shadow: 0 0 10px #38bdf8,
               0 0 20px #38bdf8;
}
.truncate {
  white-space: nowrap;
  overflow: hidden;
  text-overflow: ellipsis;
}

CSS: Flexbox & Grid

Flexbox — Container Properties
display:flex · flex-direction · flex-wrap · justify-content · align-items · gap
One-dimensional layout system — controls alignment along a single axis.
.container {
  display: flex;
  flex-direction: row;       /* row | column | row-reverse */
  flex-wrap: wrap;           /* wrap | nowrap | wrap-reverse */
  justify-content: center;   /* Main axis alignment */
  /* flex-start | flex-end | center |
     space-between | space-around | space-evenly */
  align-items: center;       /* Cross axis alignment */
  /* flex-start | flex-end | center | stretch | baseline */
  align-content: flex-start; /* Multi-line cross axis */
  gap: 16px;                 /* Gap between items */
  column-gap: 24px;
  row-gap: 12px;
}
Flexbox — Item Properties
flex-grow · flex-shrink · flex-basis · flex · align-self · order
Controls how individual flex items grow, shrink, and align.
.item {
  flex-grow: 1;       /* Grow to fill space (ratio) */
  flex-shrink: 0;     /* Don't shrink */
  flex-basis: 200px;  /* Starting size */
  flex: 1 0 200px;    /* Shorthand: grow shrink basis */

  align-self: flex-end; /* Override align-items */
  order: -1;            /* Reorder visually */
}

/* Equal columns */
.col { flex: 1; }

/* Fixed sidebar + fluid main */
.sidebar { flex: 0 0 250px; }
.main    { flex: 1; }
CSS Grid — Container Properties
display:grid · grid-template-columns/rows · gap · place-items
Two-dimensional layout — controls both rows and columns simultaneously.
.grid {
  display: grid;

  /* Define columns */
  grid-template-columns: 1fr 2fr 1fr;
  grid-template-columns: repeat(3, 1fr);
  grid-template-columns: repeat(auto-fill, minmax(250px,1fr));
  grid-template-columns: 200px auto 200px;

  /* Define rows */
  grid-template-rows: auto 1fr auto;

  /* Named areas */
  grid-template-areas:
    "header header header"
    "sidebar main main"
    "footer footer footer";

  gap: 24px;
  row-gap: 16px; column-gap: 24px;

  justify-items: center; /* Horizontal alignment */
  align-items:   center; /* Vertical alignment */
  place-items: center;   /* Shorthand */
}
CSS Grid — Item Properties
grid-column · grid-row · grid-area · justify-self · align-self
Controls placement and spanning of individual grid items.
.header  { grid-area: header; }
.sidebar { grid-area: sidebar; }
.main    { grid-area: main; }
.footer  { grid-area: footer; }

/* Manual placement */
.item {
  grid-column: 1 / 3;    /* Span from col 1 to 3 */
  grid-column: 1 / -1;   /* Span full width */
  grid-column: span 2;   /* Span 2 columns */
  grid-row: 2 / 4;       /* Span rows 2–4 */
  justify-self: end;     /* Horizontal */
  align-self: start;     /* Vertical */
}

CSS: Transitions & Animations

transition
property · duration · timing-function · delay
Smoothly animates CSS property changes on state changes like :hover.
button {
  background: #38bdf8;
  transform: scale(1);

  /* Single property */
  transition: background 0.3s ease;

  /* Multiple properties */
  transition:
    background  0.3s ease,
    transform   0.2s ease-out,
    box-shadow  0.3s ease;

  /* All properties */
  transition: all 0.3s ease;
}
button:hover {
  background: #0284c7;
  transform: scale(1.05);
}

/* Timing functions */
/* ease | linear | ease-in | ease-out |
   ease-in-out | cubic-bezier(0.4,0,0.2,1) */
animation / @keyframes
name · duration · timing · delay · iteration · direction · fill-mode · play-state
Full keyframe animations that run independently of user interaction.
/* Define animation */
@keyframes fadeIn {
  from { opacity: 0; transform: translateY(20px); }
  to   { opacity: 1; transform: translateY(0); }
}

@keyframes pulse {
  0%   { transform: scale(1); }
  50%  { transform: scale(1.1); }
  100% { transform: scale(1); }
}

@keyframes spin {
  from { transform: rotate(0deg); }
  to   { transform: rotate(360deg); }
}

/* Apply animation */
.card {
  animation: fadeIn 0.6s ease forwards;
}
.loader {
  animation: spin 1s linear infinite;
}
.badge {
  animation: pulse 2s ease-in-out infinite;
  /* name | dur | timing | delay |
     count | direction | fill-mode */
  animation: pulse 2s ease 0.5s infinite
             alternate both;
}
transform
translate · scale · rotate · skew · perspective · matrix
Applies 2D and 3D geometric transformations to elements.
.card {
  /* 2D transforms */
  transform: translateX(50px);
  transform: translateY(-20px);
  transform: translate(10px, 20px);
  transform: scale(1.5);
  transform: scaleX(2);
  transform: rotate(45deg);
  transform: skew(10deg, 5deg);

  /* Chain multiple */
  transform: translate(-50%,-50%) scale(1.1);

  /* 3D transforms */
  transform: perspective(400px) rotateY(45deg);
  transform: rotateX(15deg) rotateY(25deg);

  /* Origin point */
  transform-origin: top left;
  transform-origin: 50% 50%; /* center (default) */
}
